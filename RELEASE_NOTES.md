# Mineradio Wallpaper 发布版 1.0

首个公开发布版，包含：

- 8 个 Mineradio / Wallpaper Engine 音频响应视觉预设；
- 网易云优先、LRCLIB 备用的舞台歌词；
- Wallpaper Engine 原生媒体时间轴、暂停、切歌和 seek 同步；
- 3D 媒体历史歌单架、中文设置、性能档位和操作日志；
- 壁纸安装、备份、校验与卸载脚本；
- 用户明确确认后启用的 BetterNCM + InfLink-rs 网易云精准时间轴增强；
- Windows GitHub Actions、安装器沙盒验证、联网取词探针和 Release 完整性检查。

完整增强包含未签名 DLL 注入风险。请阅读压缩包中的 `docs/INSTALLATION.md` 与 `SECURITY.md` 后安装。
